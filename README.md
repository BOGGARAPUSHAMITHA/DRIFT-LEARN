# DRIFT-LEARN

### Concept Drift Detection System for Student Learning Behavior

DRIFT-LEARN is an intelligent learning platform that monitors student performance and detects **concept drift** using machine learning. It helps instructors understand how student learning patterns change over time and provide better guidance.

## Key Features

* Student and Instructor authentication
* Coding practice and quizzes
* Student progress tracking
* Instructor analytics dashboard
* Machine learning–based concept drift detection

## Tech Stack

**Frontend:** React.js, Next.js, Tailwind CSS, Chart.js
**Backend:** Node.js, Express.js
**Machine Learning:** Python, FastAPI, Scikit-learn, Pandas, NumPy
**Database:** MongoDB Atlas
**Authentication:** JWT, bcrypt

## Project Structure

* Frontend – Student and instructor interfaces
* Backend – APIs and server logic
* ML Service – Detects concept drift in learning data

## Installation

Clone the repository and install dependencies.

```bash
git clone https://github.com/BOGGARAPUSHAMITHA/DRIFT-LEARN.git
cd DRIFT-LEARN
npm install
npm run dev
```
